Solution Figure 7.1.1 : Triangle linéique (angle droit en O)

FIGURE 7.1.1 - Triangle linéique de côtés a et b

Préparation et Méthode : Théorème de Guldin

On cherche le centre d'inertie $G = (x_G, y_G, 0)$ d'un triangle formé de trois segments droits (corps linéique) de densité constante.

  • Géométrie :
    • Segment OA : sur l'axe Oy, de (0,0) à (0,a). Longueur $L_{OA} = a$.
    • Segment OB : sur l'axe Ox, de (0,0) à (b,0). Longueur $L_{OB} = b$.
    • Segment AB : hypoténuse, de (0,a) à (b,0). Longueur $L_{AB} = \sqrt{a^2 + b^2}$.
  • Longueur totale : $L = L_{OA} + L_{OB} + L_{AB} = a + b + \sqrt{a^2 + b^2}$
  • Théorème de Guldin (1er) : Pour trouver le centre d'inertie G, on utilise la relation entre l'aire $S_{/\Delta}$ générée par la rotation d'une ligne de longueur L autour d'un axe $(\Delta)$ et la distance $R_G$ de G à cet axe : $$S_{/\Delta} = L \times (2\pi R_G) \implies R_G = \frac{S_{/\Delta}}{2\pi L}$$
  • Application :
    • Pour $x_G$, on fait tourner la ligne autour de l'axe Oy ($R_{G, Oy} = x_G$).
    • Pour $y_G$, on fait tourner la ligne autour de l'axe Ox ($R_{G, Ox} = y_G$).

Calcul du Centre d'Inertie $G = (x_G, y_G)$

On calcule l'aire totale générée par la rotation autour de chaque axe en additionnant les aires générées par chaque segment.

1. Calcul de $x_G$ (Rotation autour de Oy)

  • Surface générée $S_{/\Delta, Oy}$ par les segments :
    • OA (sur l'axe Oy) : $S_{A,OA} = 0$.
    • OB (sur l'axe Ox) : génère un disque de rayon b $\implies S_{A,OB} = \pi b^2$.
    • AB (hypoténuse) : génère un cône (rayon b, génératrice $L_{AB}$) $\implies S_{A,AB} = \pi b L_{AB} = \pi b \sqrt{a^2+b^2}$.
  • Aire totale générée : $S_{/\Delta, Oy} = 0 + \pi b^2 + \pi b \sqrt{a^2+b^2} = \pi b (b + \sqrt{a^2+b^2})$.
  • Application de Guldin : $$ x_G = \frac{S_{/\Delta,Oy}}{2\pi L} = \frac{\pi b (b + \sqrt{a^2+b^2})}{2\pi (a + b + \sqrt{a^2+b^2})} = \frac{b (b + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

2. Calcul de $y_G$ (Rotation autour de Ox)

  • Surface générée $S_{/\Delta, Ox}$ par les segments :
    • OA (sur l'axe Oy) : génère un disque de rayon a $\implies S_{A,OA} = \pi a^2$.
    • OB (sur l'axe Ox) : $S_{A,OB} = 0$.
    • AB (hypoténuse) : génère un cône (rayon a, génératrice $L_{AB}$) $\implies S_{A,AB} = \pi a L_{AB} = \pi a \sqrt{a^2+b^2}$.
  • Aire totale générée : $S_{/\Delta, Ox} = \pi a^2 + 0 + \pi a \sqrt{a^2+b^2} = \pi a (a + \sqrt{a^2+b^2})$.
  • Application de Guldin : $$ y_G = \frac{S_{/\Delta,Ox}}{2\pi L} = \frac{\pi a (a + \sqrt{a^2+b^2})}{2\pi (a + b + \sqrt{a^2+b^2})} = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

Résultat Final

Le centre d'inertie G du triangle linéique a pour coordonnées :

$$ x_G = \frac{b (b + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$ $$ y_G = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$ $$ z_G = 0 $$

Vérification (Cas Particulier a=b)

Si $a=b$, le triangle est isocèle rectangle. G doit être sur la bissectrice ($x_G=y_G$).

Le calcul donne : $$ x_G = y_G = \frac{a(a+\sqrt{a^2+a^2})}{2(a+a+\sqrt{a^2+a^2})} = \frac{a(a+\sqrt{2a^2})}{2(2a+\sqrt{2a^2})} = \frac{a^2(1+\sqrt{2})}{2a(2+\sqrt{2})} = \frac{a(1+\sqrt{2})}{2\sqrt{2}(\sqrt{2}+1)} = \frac{a}{2\sqrt{2}} $$ Le résultat est cohérent.

Note sur une Méthode Alternative

Le calcul direct du centre d'inertie par intégration ($\vec{OG} = \frac{1}{L} \int_{(L)} \vec{OP} dl$) est possible mais plus complexe :

$$ x_G = \frac{1}{L} \left( \int_{OA} x dl + \int_{OB} x dl + \int_{AB} x dl \right) \quad ; \quad y_G = \frac{1}{L} \left( \int_{OA} y dl + \int_{OB} y dl + \int_{AB} y dl \right) $$

Le théorème de Guldin est nettement plus simple ici car les surfaces générées par rotation (disques, surfaces latérales de cônes) sont des formes géométriques standards dont les aires sont connues.
