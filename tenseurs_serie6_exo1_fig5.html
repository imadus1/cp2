FIGURE 5 - Corps composé de trois tiges dans le plan XY

Méthode requise : Additivité et Théorème de Huygens, vérification par Calcul direct.

Concepts Clés :

  • Additivité : Le tenseur total est la somme des tenseurs des parties. $[I_O]_{\text{total}} = \sum_i [I_O]_i$.
  • Théorème de Huygens : Utilisé pour chaque segment pour trouver $[I_O]_i$ à partir de $[I_{G_i}]$. $[I_O]_i = [I_{G_i}] + [I_{O/G_i}]$.
  • Plan de symétrie : Le corps est dans le plan $z = 0$, qui est un plan de symétrie matérielle. Donc, $E = I_{xz} = 0$ et $F = I_{yz} = 0$. Le produit d'inertie $D = I_{xy}$ n'est généralement pas nul car il n'y a pas de symétrie par rapport aux plans $x = 0$ ou $y = 0$.
  • Corps linéique : Chaque segment est modélisé comme un corps linéique ($dm = \lambda dl$).
  • Tenseur d'une tige inclinée par changement de base (Rotation) : Pour trouver le tenseur d'inertie d'une tige inclinée (par rapport à son centre de masse G) dans une base globale $R$, on peut d'abord l'exprimer dans une base locale $R'$ où la tige est alignée avec un axe (par exemple, l'axe x'). Dans cette base locale, le tenseur $[I_G]_{R'}$ a une forme diagonale simple, par exemple $[I_G]_{R'} = \text{diag}(0, ML^2/12, ML^2/12)$ si la tige est sur x'. On utilise ensuite la formule de changement de base pour les tenseurs d'ordre 2 afin d'obtenir le tenseur dans la base globale $R$. Si $P$ est la matrice de passage de $R$ à $R'$ (colonnes : nouveaux vecteurs de base de $R'$ exprimés dans l'ancienne base $R$), alors $[I_G]_R = P [I_G]_{R'} P^T$. La matrice $P$ est orthogonale ($P^T P = P P^T = \mathbf{1}$, donc $P^T = P^{-1}$). Cette méthode est illustrée pour le Segment 3.

Géométrie et Décomposition :

Le corps est formé de 3 segments de tiges homogènes (densité $\lambda$) dans le plan $z = 0$.

  • Seg 1 (AB) : de $A = \begin{pmatrix} 0 \\ a \\ 0 \end{pmatrix}$ à $B = \begin{pmatrix} a \\ a \\ 0 \end{pmatrix}$.
    • Orientation : Parallèle à OX.
    • Longueur $L_1 = a$.
    • Masse $M_1 = \lambda a$.
    • Centre de masse $G_1 = (A+B)/2 = \begin{pmatrix} a/2 \\ a \\ 0 \end{pmatrix}$.
  • Seg 2 (BC) : de $B = \begin{pmatrix} a \\ a \\ 0 \end{pmatrix}$ à $C = \begin{pmatrix} a \\ 2a \\ 0 \end{pmatrix}$.
    • Orientation : Parallèle à OY.
    • Longueur $L_2 = a$.
    • Masse $M_2 = \lambda a$.
    • Centre de masse $G_2 = (B+C)/2 = \begin{pmatrix} a \\ 3a/2 \\ 0 \end{pmatrix}$.
  • Seg 3 (CD) : de $C = \begin{pmatrix} a \\ 2a \\ 0 \end{pmatrix}$ à $D = \begin{pmatrix} a(1-\sqrt{2}) \\ a(2+\sqrt{2}) \\ 0 \end{pmatrix}$.
    • Vecteur $\vec{CD} = D - C = \begin{pmatrix} -a\sqrt{2} \\ a\sqrt{2} \\ 0 \end{pmatrix}$.
    • Longueur $L_3 = ||\vec{CD}|| = \sqrt{(-a\sqrt{2})^2 + (a\sqrt{2})^2 + 0^2} = \sqrt{2a^2 + 2a^2} = 2a$.
    • Masse $M_3 = \lambda L_3 = 2\lambda a$.
    • Centre de masse $G_3 = (C+D)/2 = \begin{pmatrix} (a + a - a\sqrt{2})/2 \\ (2a + 2a + a\sqrt{2})/2 \\ 0 \end{pmatrix} = \begin{pmatrix} a(1 - \sqrt{2}/2) \\ a(2 + \sqrt{2}/2) \\ 0 \end{pmatrix}$.
    • Vecteur unitaire $\vec{u}_3 = \vec{CD}/L_3 = \begin{pmatrix} -1/\sqrt{2} \\ 1/\sqrt{2} \\ 0 \end{pmatrix}$. (Angle de 135° avec OX).
  • Masse totale : $M = M_1 + M_2 + M_3 = \lambda a + \lambda a + 2\lambda a = 4\lambda a$.

Méthode 1 : Additivité et Théorème de Huygens

On calcule $[I_O]_i = [I_{G_i}] + [I_{O/G_i}]$ pour chaque segment $i = 1, 2, 3$.

Segment 1 (AB) : Parallèle à OX

  • $M_1 = \lambda a, L_1 = a, G_1 = (a/2, a, 0)$.
  • Tenseur $[I_{G1}]$ (cf. Fig 1.1) : $$ [I_{G1}] = \begin{pmatrix} 0 & 0 & 0 \\ 0 & M_1 a^2 / 12 & 0 \\ 0 & 0 & M_1 a^2 / 12 \end{pmatrix} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & \lambda a^3 / 12 & 0 \\ 0 & 0 & \lambda a^3 / 12 \end{pmatrix} $$
  • Transport $[I_{O/G1}]$ : $x_G=a/2, y_G=a, z_G=0$. $$ [I_{O/G1}] = M_1 \begin{pmatrix} y_G^2+z_G^2 & -x_G y_G & -x_G z_G \\ -x_G y_G & x_G^2+z_G^2 & -y_G z_G \\ -x_G z_G & -y_G z_G & x_G^2+y_G^2 \end{pmatrix} = \lambda a \begin{pmatrix} a^2+0 & -(a/2)a & 0 \\ -(a/2)a & (a/2)^2+0 & 0 \\ 0 & 0 & (a/2)^2+a^2 \end{pmatrix} = \lambda a \begin{pmatrix} a^2 & -a^2/2 & 0 \\ -a^2/2 & a^2/4 & 0 \\ 0 & 0 & 5a^2/4 \end{pmatrix} = \begin{pmatrix} \lambda a^3 & -\lambda a^3/2 & 0 \\ -\lambda a^3/2 & \lambda a^3/4 & 0 \\ 0 & 0 & 5\lambda a^3/4 \end{pmatrix} $$
  • Tenseur $[I_O]_1$: $$ [I_O]_1 = [I_{G1}] + [I_{O/G1}] = \begin{pmatrix} \lambda a^3 & -\lambda a^3/2 & 0 \\ -\lambda a^3/2 & \lambda a^3/12 + \lambda a^3/4 & 0 \\ 0 & 0 & \lambda a^3/12 + 5\lambda a^3/4 \end{pmatrix} = \begin{pmatrix} \lambda a^3 & -\lambda a^3/2 & 0 \\ -\lambda a^3/2 & \lambda a^3/3 & 0 \\ 0 & 0 & 4\lambda a^3/3 \end{pmatrix} $$

Segment 2 (BC) : Parallèle à OY

  • $M_2 = \lambda a, L_2 = a, G_2 = (a, 3a/2, 0)$.
  • Tenseur $[I_{G2}]$ (tige // OY) : $$ [I_{G2}] = \begin{pmatrix} M_2 a^2 / 12 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & M_2 a^2 / 12 \end{pmatrix} = \begin{pmatrix} \lambda a^3 / 12 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & \lambda a^3 / 12 \end{pmatrix} $$
  • Transport $[I_{O/G2}]$ : $x_G=a, y_G=3a/2, z_G=0$. $$ [I_{O/G2}] = M_2 \begin{pmatrix} y_G^2+z_G^2 & -x_G y_G & -x_G z_G \\ -x_G y_G & x_G^2+z_G^2 & -y_G z_G \\ -x_G z_G & -y_G z_G & x_G^2+y_G^2 \end{pmatrix} = \lambda a \begin{pmatrix} (3a/2)^2+0 & -a(3a/2) & 0 \\ -a(3a/2) & a^2+0 & 0 \\ 0 & 0 & a^2+(3a/2)^2 \end{pmatrix} = \lambda a \begin{pmatrix} 9a^2/4 & -3a^2/2 & 0 \\ -3a^2/2 & a^2 & 0 \\ 0 & 0 & 13a^2/4 \end{pmatrix} = \begin{pmatrix} 9\lambda a^3/4 & -3\lambda a^3/2 & 0 \\ -3\lambda a^3/2 & \lambda a^3 & 0 \\ 0 & 0 & 13\lambda a^3/4 \end{pmatrix} $$
  • Tenseur $[I_O]_2$: $$ [I_O]_2 = [I_{G2}] + [I_{O/G2}] = \begin{pmatrix} \lambda a^3/12 + 9\lambda a^3/4 & -3\lambda a^3/2 & 0 \\ -3\lambda a^3/2 & \lambda a^3 & 0 \\ 0 & 0 & \lambda a^3/12 + 13\lambda a^3/4 \end{pmatrix} = \begin{pmatrix} 7\lambda a^3/3 & -3\lambda a^3/2 & 0 \\ -3\lambda a^3/2 & \lambda a^3 & 0 \\ 0 & 0 & 10\lambda a^3/3 \end{pmatrix} $$

Segment 3 (CD) : Orienté à 135° - Méthode par Rotation

Nous calculons le tenseur d'inertie $[I_O]_3$ pour le segment CD en utilisant la rotation de base et le théorème de Huygens.

Caractéristiques du segment CD

  • Extrémités : $C = (a, 2a, 0)$, $D = (a(1 - \sqrt{2}), a(2 + \sqrt{2}), 0)$
  • Longueur : $L_3 = ||\vec{CD}|| = ||(−a\sqrt{2}, a\sqrt{2}, 0)|| = \sqrt{2a^2 + 2a^2} = 2a$
  • Masse : $M_3 = \lambda L_3 = 2\lambda a$
  • Centre de masse : $G_3 = (C+D)/2 = (a(1 - \frac{\sqrt{2}}{2}), a(2 + \frac{\sqrt{2}}{2}), 0)$
  • Vecteur directeur unitaire (de C vers D) : $\vec{u}_3 = \frac{\vec{CD}}{L_3} = \frac{1}{2a}(-a\sqrt{2}, a\sqrt{2}, 0) = (-\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}, 0)$. Ceci correspond à un angle de 135° par rapport à l'axe Ox.

Étape 1 : Tenseur d'inertie dans un repère local en G3 ($R'$)

Considérons un repère local $R'(G_3, \vec{i}', \vec{j}', \vec{k}')$ où $\vec{i}' = \vec{u}_3$ est aligné avec la tige, $\vec{k}' = \vec{k}$ (axe Oz), et $\vec{j}' = \vec{k}' \times \vec{i}' = (0, 0, 1) \times (-\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}, 0) = (-\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}, 0)$. Dans ce repère local $R'$, la tige est alignée avec l'axe $G_3x'$. Pour une tige mince de masse $M$ et longueur $L$ centrée à l'origine le long de l'axe x', son tenseur d'inertie est :

$$ [I_{G3}]_{R'} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & M_3 L_3^2 / 12 & 0 \\ 0 & 0 & M_3 L_3^2 / 12 \end{pmatrix} $$ Avec $M_3 = 2\lambda a$ et $L_3 = 2a$: $$ \frac{M_3 L_3^2}{12} = \frac{(2\lambda a)(2a)^2}{12} = \frac{2\lambda a \cdot 4a^2}{12} = \frac{8\lambda a^3}{12} = \frac{2\lambda a^3}{3} $$ Donc, $$ [I_{G3}]_{R'} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 2\lambda a^3 / 3 & 0 \\ 0 & 0 & 2\lambda a^3 / 3 \end{pmatrix} $$

Étape 2 : Matrice de Rotation (Matrice de passage de $R$ vers $R'$)

La matrice de passage $P$ du repère global $R(O,\vec{i},\vec{j},\vec{k})$ au repère local $R'(G_3,\vec{i}',\vec{j}',\vec{k}')$ a pour colonnes les vecteurs de base de $R'$ exprimés dans $R$.

$$ P = [\vec{i}' | \vec{j}' | \vec{k}']_R = \begin{pmatrix} -1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 0 & 0 & 1 \end{pmatrix} $$

Cette matrice est orthogonale ($P^T = P^{-1}$).

Étape 3 : Rotation du Tenseur vers le Repère Global (en G3)

La formule de changement de base pour le tenseur d'inertie $[I]$ d'une base $R'$ vers une base $R$ est $[I]_R = P [I]_{R'} P^T$, où $P$ est la matrice de passage de $R$ à $R'$.

Calculons le produit: $$ [I_{G3}]_R = \begin{pmatrix} -1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 0 & 0 & 1 \end{pmatrix} \begin{pmatrix} 0 & 0 & 0 \\ 0 & \frac{2\lambda a^3}{3} & 0 \\ 0 & 0 & \frac{2\lambda a^3}{3} \end{pmatrix} \begin{pmatrix} -1/\sqrt{2} & 1/\sqrt{2} & 0 \\ -1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 0 & 0 & 1 \end{pmatrix} $$ produit 1 : $$ \begin{pmatrix} -1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 0 & 0 & 1 \end{pmatrix} \begin{pmatrix} 0 & 0 & 0 \\ 0 & \frac{2\lambda a^3}{3} & 0 \\ 0 & 0 & \frac{2\lambda a^3}{3} \end{pmatrix} = \begin{pmatrix} 0 & -\frac{1}{\sqrt{2}}\frac{2\lambda a^3}{3} & 0 \\ 0 & -\frac{1}{\sqrt{2}}\frac{2\lambda a^3}{3} & 0 \\ 0 & 0 & \frac{2\lambda a^3}{3} \end{pmatrix} = \begin{pmatrix} 0 & -\frac{\sqrt{2}\lambda a^3}{3} & 0 \\ 0 & -\frac{\sqrt{2}\lambda a^3}{3} & 0 \\ 0 & 0 & \frac{2\lambda a^3}{3} \end{pmatrix} $$ produit 2: $$ \begin{pmatrix} 0 & -\frac{\sqrt{2}\lambda a^3}{3} & 0 \\ 0 & -\frac{\sqrt{2}\lambda a^3}{3} & 0 \\ 0 & 0 & \frac{2\lambda a^3}{3} \end{pmatrix} \begin{pmatrix} -1/\sqrt{2} & 1/\sqrt{2} & 0 \\ -1/\sqrt{2} & -1/\sqrt{2} & 0 \\ 0 & 0 & 1 \end{pmatrix} $$ $$ = \begin{pmatrix} 0 + (-\frac{\sqrt{2}\lambda a^3}{3})(-\frac{1}{\sqrt{2}}) + 0 & 0 + (-\frac{\sqrt{2}\lambda a^3}{3})(-\frac{1}{\sqrt{2}}) + 0 & 0 \\ 0 + (-\frac{\sqrt{2}\lambda a^3}{3})(-\frac{1}{\sqrt{2}}) + 0 & 0 + (-\frac{\sqrt{2}\lambda a^3}{3})(-\frac{1}{\sqrt{2}}) + 0 & 0 \\ 0 & 0 & \frac{2\lambda a^3}{3} \end{pmatrix} $$ $$ = \begin{pmatrix} \frac{2\lambda a^3}{3 \cdot 2} & \frac{2\lambda a^3}{3 \cdot 2} & 0 \\ \frac{2\lambda a^3}{3 \cdot 2} & \frac{2\lambda a^3}{3 \cdot 2} & 0 \\ 0 & 0 & \frac{2\lambda a^3}{3} \end{pmatrix} = \lambda a^3 \begin{pmatrix} 1/3 & 1/3 & 0 \\ 1/3 & 1/3 & 0 \\ 0 & 0 & 2/3 \end{pmatrix} $$

Le résultat pour $[I_{G3}]_R$ .

$$ [I_{G3}]_R = \lambda a^3 \begin{pmatrix} 1/3 & 1/3 & 0 \\ 1/3 & 1/3 & 0 \\ 0 & 0 & 2/3 \end{pmatrix} $$

Ce $[I_{G3}]_R$ est le tenseur d'inertie de la tige par rapport à son centre de masse G3, mais exprimé dans la base globale $(\vec{i}, \vec{j}, \vec{k})$.

Étape 4 : Terme de Huygens

Le terme de transport $[I_{O/G3}]$ est le tenseur d'inertie d'une masse ponctuelle $M_3$ située en $G_3$, par rapport à O. Avec $x_G = a(1 - \frac{\sqrt{2}}{2})$, $y_G = a(2 + \frac{\sqrt{2}}{2})$, $z_G = 0$.

  • $y_G^2 + z_G^2 = a^2(2 + \frac{\sqrt{2}}{2})^2 = a^2(4 + 2\sqrt{2} + 1/2) = a^2(4.5 + 2\sqrt{2})$
  • $x_G^2 + z_G^2 = a^2(1 - \frac{\sqrt{2}}{2})^2 = a^2(1 - \sqrt{2} + 1/2) = a^2(1.5 - \sqrt{2})$
  • $x_G^2 + y_G^2 = a^2(1.5 - \sqrt{2}) + a^2(4.5 + 2\sqrt{2}) = a^2(6 + \sqrt{2})$
  • $-x_G y_G = -a(1 - \frac{\sqrt{2}}{2}) a(2 + \frac{\sqrt{2}}{2}) = -a^2(2 + \frac{\sqrt{2}}{2} - \sqrt{2} - \frac{1}{2}) = -a^2(1.5 - \frac{\sqrt{2}}{2})$
$$ [I_{O/G3}] = M_3 \begin{pmatrix} y_G^2+z_G^2 & -x_G y_G & -x_G z_G \\ -x_G y_G & x_G^2+z_G^2 & -y_G z_G \\ -x_G z_G & -y_G z_G & x_G^2+y_G^2 \end{pmatrix} $$ $$ = 2\lambda a \begin{pmatrix} a^2(4.5 + 2\sqrt{2}) & -a^2(1.5 - \frac{\sqrt{2}}{2}) & 0 \\ -a^2(1.5 - \frac{\sqrt{2}}{2}) & a^2(1.5 - \sqrt{2}) & 0 \\ 0 & 0 & a^2(6 + \sqrt{2}) \end{pmatrix} $$ $$ = \lambda a^3 \begin{pmatrix} 9 + 4\sqrt{2} & -3 + \sqrt{2} & 0 \\ -3 + \sqrt{2} & 3 - 2\sqrt{2} & 0 \\ 0 & 0 & 12 + 2\sqrt{2} \end{pmatrix} $$

Étape 5 : Tenseur d'Inertie Final pour CD en O

$$ [I_O]_3 = [I_{G3}]_R + [I_{O/G3}] $$ $$ = \lambda a^3 \begin{pmatrix} 1/3 & 1/3 & 0 \\ 1/3 & 1/3 & 0 \\ 0 & 0 & 2/3 \end{pmatrix} + \lambda a^3 \begin{pmatrix} 9 + 4\sqrt{2} & -3 + \sqrt{2} & 0 \\ -3 + \sqrt{2} & 3 - 2\sqrt{2} & 0 \\ 0 & 0 & 12 + 2\sqrt{2} \end{pmatrix} $$ $$ = \lambda a^3 \begin{pmatrix} \frac{1}{3} + 9 + 4\sqrt{2} & \frac{1}{3} - 3 + \sqrt{2} & 0 \\ \frac{1}{3} - 3 + \sqrt{2} & \frac{1}{3} + 3 - 2\sqrt{2} & 0 \\ 0 & 0 & \frac{2}{3} + 12 + 2\sqrt{2} \end{pmatrix} $$ $$ = \lambda a^3 \begin{pmatrix} \frac{28}{3} + 4\sqrt{2} & -\frac{8}{3} + \sqrt{2} & 0 \\ -\frac{8}{3} + \sqrt{2} & \frac{10}{3} - 2\sqrt{2} & 0 \\ 0 & 0 & \frac{38}{3} + 2\sqrt{2} \end{pmatrix} $$

Tenseur Total :

$[I_O] = [I_O]_1 + [I_O]_2 + [I_O]_3 $.

$$ [I_O] = \begin{pmatrix} \lambda a^3 & -\lambda a^3/2 & 0 \\ -\lambda a^3/2 & \lambda a^3/3 & 0 \\ 0 & 0 & 4\lambda a^3/3 \end{pmatrix} + \begin{pmatrix} 7\lambda a^3/3 & -3\lambda a^3/2 & 0 \\ -3\lambda a^3/2 & \lambda a^3 & 0 \\ 0 & 0 & 10\lambda a^3/3 \end{pmatrix} + \lambda a^3 \begin{pmatrix} \frac{28}{3} + 4\sqrt{2} & -\frac{8}{3} + \sqrt{2} & 0 \\ -\frac{8}{3} + \sqrt{2} & \frac{10}{3} - 2\sqrt{2} & 0 \\ 0 & 0 & \frac{38}{3} + 2\sqrt{2} \end{pmatrix} $$

Calcul des composantes A, B, D, C :

  • $A = A_1 + A_2 + A_3 = \lambda a^3 + \frac{7\lambda a^3}{3} + \lambda a^3(\frac{28}{3} + 4\sqrt{2}) = \lambda a^3 (\frac{3+7+28}{3} + 4\sqrt{2}) = \lambda a^3 (\frac{38}{3} + 4\sqrt{2})$
  • $B = B_1 + B_2 + B_3 = \frac{\lambda a^3}{3} + \lambda a^3 + \lambda a^3(\frac{10}{3} - 2\sqrt{2}) = \lambda a^3 (\frac{1+3+10}{3} - 2\sqrt{2}) = \lambda a^3 (\frac{14}{3} - 2\sqrt{2})$
  • $-D = (-D)_1 + (-D)_2 + (-D)_3 = -\frac{\lambda a^3}{2} - \frac{3\lambda a^3}{2} + \lambda a^3(-\frac{8}{3} + \sqrt{2}) = \lambda a^3 (-\frac{4}{2} - \frac{8}{3} + \sqrt{2}) = \lambda a^3 (-2 - \frac{8}{3} + \sqrt{2}) = \lambda a^3 (\frac{-6-8}{3} + \sqrt{2}) = \lambda a^3 (-\frac{14}{3} + \sqrt{2})$
    $ \implies D = \lambda a^3 (\frac{14}{3} - \sqrt{2}) $
  • $C = C_1 + C_2 + C_3 = \frac{4\lambda a^3}{3} + \frac{10\lambda a^3}{3} + \lambda a^3(\frac{38}{3} + 2\sqrt{2}) = \lambda a^3 (\frac{4+10+38}{3} + 2\sqrt{2}) = \lambda a^3 (\frac{52}{3} + 2\sqrt{2})$

Vérification C = A + B :

$$ A + B = \lambda a^3 (\frac{38}{3} + 4\sqrt{2}) + \lambda a^3 (\frac{14}{3} - 2\sqrt{2}) = \lambda a^3 (\frac{38+14}{3} + (4-2)\sqrt{2}) = \lambda a^3 (\frac{52}{3} + 2\sqrt{2}) = C $$

OK (objet plan).

Résultat (Huygens) :

$$ [I_O]_{\text{Huygens}} = \lambda a^3 \begin{pmatrix} \frac{38}{3} + 4\sqrt{2} & -(\frac{14}{3} - \sqrt{2}) & 0 \\ -(\frac{14}{3} - \sqrt{2}) & \frac{14}{3} - 2\sqrt{2} & 0 \\ 0 & 0 & \frac{52}{3} + 2\sqrt{2} \end{pmatrix} $$

Méthode 2 : Calcul direct par intégration

Calculons $A = \int (y^2 + z^2)dm$, $B = \int (x^2 + z^2)dm$, $D = \int xy dm$. ($z=0, E=F=0$). $dm = \lambda dl$.

Segment 1 (AB): $y=a, x \in [0, a]$, $dl = dx$. $dm = \lambda dx$.

$$ A_1 = \int_0^a (a^2 + 0^2)\lambda dx = \lambda a^2[x]_0^a = \lambda a^3 $$ $$ B_1 = \int_0^a (x^2 + 0^2)\lambda dx = \lambda [x^3/3]_0^a = \lambda a^3/3 $$ $$ D_1 = \int_0^a x(a)\lambda dx = \lambda a [x^2/2]_0^a = \lambda a^3/2 $$

Segment 2 (BC): $x=a, y \in [a, 2a]$, $dl = dy$. $dm = \lambda dy$.

$$ A_2 = \int_a^{2a} (y^2 + 0^2)\lambda dy = \lambda [y^3/3]_a^{2a} = \lambda (\frac{8a^3}{3} - \frac{a^3}{3}) = 7\lambda a^3/3 $$ $$ B_2 = \int_a^{2a} (a^2 + 0^2)\lambda dy = \lambda a^2 [y]_a^{2a} = \lambda a^2 (2a - a) = \lambda a^3 $$ $$ D_2 = \int_a^{2a} a(y)\lambda dy = \lambda a [y^2/2]_a^{2a} = \lambda a (\frac{4a^2}{2} - \frac{a^2}{2}) = 3\lambda a^3/2 $$

Segment 3 (CD): Paramétrisation par l'abscisse curviligne $s \in [0, 2a]$ depuis C.

$ P(s) = C + s \vec{u}_3 = \begin{pmatrix} a \\ 2a \\ 0 \end{pmatrix} + s \begin{pmatrix} -1/\sqrt{2} \\ 1/\sqrt{2} \\ 0 \end{pmatrix} = \begin{pmatrix} a - s/\sqrt{2} \\ 2a + s/\sqrt{2} \\ 0 \end{pmatrix} $. $x(s) = a - s/\sqrt{2}$, $y(s) = 2a + s/\sqrt{2}$. $dm = \lambda ds$.

$$ A_3 = \int_0^{2a} y(s)^2 \lambda ds = \lambda \int_0^{2a} (2a + s/\sqrt{2})^2 ds = \lambda \int_0^{2a} (4a^2 + \frac{4as}{\sqrt{2}} + \frac{s^2}{2}) ds $$ $$ = \lambda \left[ 4a^2s + \frac{4a s^2}{2\sqrt{2}} + \frac{s^3}{6} \right]_0^{2a} = \lambda \left( 8a^3 + \frac{16a^3}{2\sqrt{2}} + \frac{8a^3}{6} \right) = \lambda \left( 8a^3 + \frac{8a^3}{\sqrt{2}} + \frac{4a^3}{3} \right) = \lambda a^3 \left( \frac{24+4}{3} + 4\sqrt{2} \right) = \lambda a^3 \left( \frac{28}{3} + 4\sqrt{2} \right) $$ $$ B_3 = \int_0^{2a} x(s)^2 \lambda ds = \lambda \int_0^{2a} (a - s/\sqrt{2})^2 ds = \lambda \int_0^{2a} (a^2 - \frac{2as}{\sqrt{2}} + \frac{s^2}{2}) ds $$ $$ = \lambda \left[ a^2s - \frac{2a s^2}{2\sqrt{2}} + \frac{s^3}{6} \right]_0^{2a} = \lambda \left( 2a^3 - \frac{4a^3}{\sqrt{2}} + \frac{8a^3}{6} \right) = \lambda \left( 2a^3 - 2\sqrt{2}a^3 + \frac{4a^3}{3} \right) = \lambda a^3 \left( \frac{6+4}{3} - 2\sqrt{2} \right) = \lambda a^3 \left( \frac{10}{3} - 2\sqrt{2} \right) $$ $$ D_3 = \int_0^{2a} x(s)y(s) \lambda ds = \lambda \int_0^{2a} (a - s/\sqrt{2})(2a + s/\sqrt{2}) ds = \lambda \int_0^{2a} (2a^2 + \frac{as}{\sqrt{2}} - \frac{2as}{\sqrt{2}} - \frac{s^2}{2}) ds = \lambda \int_0^{2a} (2a^2 - \frac{as}{\sqrt{2}} - \frac{s^2}{2}) ds $$ $$ = \lambda \left[ 2a^2s - \frac{a s^2}{2\sqrt{2}} - \frac{s^3}{6} \right]_0^{2a} = \lambda \left( 4a^3 - \frac{4a^3}{2\sqrt{2}} - \frac{8a^3}{6} \right) = \lambda \left( 4a^3 - \sqrt{2}a^3 - \frac{4a^3}{3} \right) = \lambda a^3 \left( \frac{12-4}{3} - \sqrt{2} \right) = \lambda a^3 \left( \frac{8}{3} - \sqrt{2} \right) $$

Somme des intégrales :

$$ A = A_1 + A_2 + A_3 = \lambda a^3 + \frac{7\lambda a^3}{3} + \lambda a^3(\frac{28}{3} + 4\sqrt{2}) = \lambda a^3(1 + \frac{7}{3} + \frac{28}{3} + 4\sqrt{2}) = \lambda a^3(\frac{3+7+28}{3} + 4\sqrt{2}) = \lambda a^3(\frac{38}{3} + 4\sqrt{2}) $$ $$ B = B_1 + B_2 + B_3 = \frac{\lambda a^3}{3} + \lambda a^3 + \lambda a^3(\frac{10}{3} - 2\sqrt{2}) = \lambda a^3(\frac{1}{3} + 1 + \frac{10}{3} - 2\sqrt{2}) = \lambda a^3(\frac{1+3+10}{3} - 2\sqrt{2}) = \lambda a^3(\frac{14}{3} - 2\sqrt{2}) $$ $$ D = D_1 + D_2 + D_3 = \frac{\lambda a^3}{2} + \frac{3\lambda a^3}{2} + \lambda a^3(\frac{8}{3} - \sqrt{2}) = \lambda a^3(\frac{1}{2} + \frac{3}{2} + \frac{8}{3} - \sqrt{2}) = \lambda a^3(2 + \frac{8}{3} - \sqrt{2}) = \lambda a^3(\frac{6+8}{3} - \sqrt{2}) = \lambda a^3(\frac{14}{3} - \sqrt{2}) $$

$C = A + B = \lambda a^3 (\frac{38}{3} + 4\sqrt{2}) + \lambda a^3 (\frac{14}{3} - 2\sqrt{2}) = \lambda a^3 (\frac{38+14}{3} + (4-2)\sqrt{2}) = \lambda a^3 (\frac{52}{3} + 2\sqrt{2})$ (Vérifié).

$E = 0, F = 0$.

Résultat :

Le tenseur d'inertie au point O est :

$$ [I_O]_{1.5} = \lambda a^3 \begin{pmatrix} \frac{38}{3} + 4\sqrt{2} & -(\frac{14}{3} - \sqrt{2}) & 0 \\ -(\frac{14}{3} - \sqrt{2}) & \frac{14}{3} - 2\sqrt{2} & 0 \\ 0 & 0 & \frac{52}{3} + 2\sqrt{2} \end{pmatrix} $$

où $\lambda$ est la densité linéique de masse et $a$ est une longueur caractéristique.
