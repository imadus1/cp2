Solution Figure 7.1.3 : Triangle linéique (angle droit en B)

FIGURE 7.1.3 - Triangle linéique O(0,0), B(b,0), A(b,a)

Méthode et Préparation (Théorème de Guldin)

On cherche le centre d'inertie $G = (x_G, y_G, 0)$ de ce triangle linéique par le premier théorème de Guldin. Rappel : $R_G = S_{/\Delta} / (2\pi L)$, où $R_G$ est la distance de G à l'axe de rotation $\Delta$, $L$ la longueur totale, et $S_{/\Delta}$ l'aire générée par la rotation.

Géométrie : Segments OB (longueur b), BA (longueur a), OA (longueur $\sqrt{a^2 + b^2}$).
Longueur totale : $L = a + b + \sqrt{a^2 + b^2}$.

Calcul du Centre d'Inertie $G = (x_G, y_G)$ par Guldin

1. Calcul de $x_G$ (Rotation autour de Oy)

Aires générées par rotation autour de Oy ($R_G = x_G$) :

  • OB (disque) : $S_{OB} = \pi b^2$
  • BA (cylindre) : $S_{BA} = 2\pi ab$
  • OA (cône) : $S_{OA} = \pi b \sqrt{a^2+b^2}$

Aire totale : $S_{/\Delta,Oy} = S_{OB} + S_{BA} + S_{OA} = \pi b (b + 2a + \sqrt{a^2+b^2})$.

Application de Guldin : $$ x_G = \frac{S_{/\Delta,Oy}}{2\pi L} = \frac{\pi b (b + 2a + \sqrt{a^2+b^2})}{2\pi (a + b + \sqrt{a^2+b^2})} = \frac{b (b + 2a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

2. Calcul de $y_G$ (Rotation autour de Ox)

Aires générées par rotation autour de Ox ($R_G = y_G$) :

  • OB (sur l'axe) : $S_{OB} = 0$
  • BA (disque) : $S_{BA} = \pi a^2$
  • OA (cône) : $S_{OA} = \pi a \sqrt{a^2+b^2}$

Aire totale : $S_{/\Delta,Ox} = S_{OB} + S_{BA} + S_{OA} = \pi a (a + \sqrt{a^2+b^2})$.

Application de Guldin : $$ y_G = \frac{S_{/\Delta,Ox}}{2\pi L} = \frac{\pi a (a + \sqrt{a^2+b^2})}{2\pi (a + b + \sqrt{a^2+b^2})} = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

Résultat Final (par Guldin)

Le centre d'inertie G a pour coordonnées :

$$ x_G = \frac{b (b + 2a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$ $$ y_G = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$ $$ z_G = 0 $$

Alternative : Utilisation de la Symétrie (par rapport à l'Ex 7.1.1)

Cette figure (triangle O(0,0), B(b,0), A(b,a)) est l'image de la figure 7.1.1 (triangle O'(0,0), B'(b,0), A'(0,a)) par réflexion autour de l'axe vertical $x=b/2$.

Le centre d'inertie $G_1=(x_{G1}, y_{G1})$ de la Fig 7.1.1 est (voir solution Ex 7.1.1) :

$$ x_{G1} = \frac{b (b + \sqrt{a^2+b^2})}{2 L}, \quad y_{G1} = \frac{a (a + \sqrt{a^2+b^2})}{2 L} \quad \left(L = a+b+\sqrt{a^2+b^2}\right) $$

Le centre d'inertie $G_3=(x_{G3}, y_{G3})$ de cette figure (Fig 7.1.3) est l'image de $G_1$ par la réflexion. Ainsi :

$$ x_{G3} = b - x_{G1} \quad \text{et} \quad y_{G3} = y_{G1} $$

Calcul de $x_{G3}$ :

$$ x_{G3} = b - \frac{b (b + \sqrt{a^2+b^2})}{2 L} = \frac{2bL - b(b + \sqrt{a^2+b^2})}{2L} $$ $$ x_{G3} = \frac{2b(a+b+\sqrt{a^2+b^2}) - b^2 - b\sqrt{a^2+b^2}}{2L} $$ $$ x_{G3} = \frac{2ab + 2b^2 + 2b\sqrt{a^2+b^2} - b^2 - b\sqrt{a^2+b^2}}{2L} = \frac{b (2a + b + \sqrt{a^2+b^2})}{2 L} $$

Donc :

$$ x_{G3} = \frac{b (b + 2a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} \quad \text{et} \quad y_{G3} = y_{G1} = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

Ces coordonnées $(x_{G3}, y_{G3})$ obtenues par symétrie sont identiques à $(x_G, y_G)$ calculées par Guldin, confirmant le résultat.
