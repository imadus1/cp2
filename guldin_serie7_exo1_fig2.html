Solution Figure 7.1.2 : Triangle linéique décalé en x

FIGURE 7.1.2 - Triangle linéique décalé en x de c

Approche et Concepts Clés

On cherche le centre d'inertie $G = (x_G, y_G, 0)$ de ce triangle linéique. Ce triangle est identique à celui de la Figure 7.1.1 (Exercice 4.1), mais translaté horizontalement de $c$. Deux méthodes sont possibles :

  • Méthode 1 : Translation (Préférable) - Utiliser le résultat de l'exercice précédent et appliquer la propriété de translation du centre d'inertie. Si un corps est translaté par $\vec{v}$, son centre d'inertie est translaté par le même $\vec{v}$. Ici, $\vec{v} = c \vec{i}$.
  • Méthode 2 : Théorème de Guldin - Appliquer directement le premier théorème de Guldin ($R_G = S_{/\Delta} / (2\pi L)$), comme dans l'exercice précédent, mais avec la nouvelle géométrie.

Géométrie et Longueur Totale

Les sommets sont C=(c,0), A=(c,a), B=(c+b,0).

  • Segment CA (vertical) : Longueur $L_{CA} = a$.
  • Segment CB (horizontal) : Longueur $L_{CB} = b$.
  • Segment AB (hypoténuse) : Longueur $L_{AB} = \sqrt{a^2 + b^2}$.
  • Longueur totale : $L = L_{CA} + L_{CB} + L_{AB} = a + b + \sqrt{a^2 + b^2}$ (identique à l'Ex 4.1).

Méthode 1 : Utilisation de la Translation (Recommandée)

Soit $G_1 = (x_{G1}, y_{G1})$ le centre d'inertie du triangle non décalé (Figure 7.1.1) :

$$ x_{G1} = \frac{b (b + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})}, \quad y_{G1} = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

Le triangle de la Figure 7.1.2 est obtenu en translatant celui de la Figure 7.1.1 de $c$ selon l'axe des x ($\vec{v} = c\vec{i}$). Le nouveau centre d'inertie $G = (x_G, y_G)$ est donc :

$$ x_G = x_{G1} + c = c + \frac{b (b + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$ $$ y_G = y_{G1} + 0 = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

Cette méthode est directe et évite de nouveaux calculs complexes.

Méthode 2 : Application Directe du Théorème de Guldin

Cette méthode recalcule tout, servant de vérification.

Calcul de $x_G$ (Rotation autour de Oy)

  • Surfaces générées par rotation autour de l'axe Oy:
    • CA : Cylindre (rayon c, hauteur a) $\implies S_{CA} = 2\pi c a$.
    • CB : Anneau (rayons c et c+b) $\implies S_{CB} = \pi(c+b)^2 - \pi c^2 = \pi(2cb + b^2)$.
    • AB : Tronc de cône (rayons c et c+b, génératrice $L_{AB}$) $\implies S_{AB} = \pi(c + (c+b)) L_{AB} = \pi(2c+b)\sqrt{a^2+b^2}$.
  • Aire totale générée : $S_{/\Delta, Oy} = S_{CA} + S_{CB} + S_{AB} = 2\pi c a + \pi b(2c + b) + \pi(2c+b)\sqrt{a^2+b^2}$.
  • Application de Guldin : $$ x_G = \frac{S_{/\Delta,Oy}}{2\pi L} = \frac{2\pi c a + \pi b(2c + b) + \pi(2c+b)\sqrt{a^2+b^2}}{2\pi (a + b + \sqrt{a^2+b^2})} $$

    Note : Ce résultat est plus complexe à écrire mais, après simplification (non détaillée ici), il est égal à $c + x_{G1}$.

Calcul de $y_G$ (Rotation autour de Ox)

  • Surfaces générées par rotation autour de l'axe Ox :
    • CA : Anneau (rayon a) $\implies S_{CA} = \pi a^2$.
    • CB (sur l'axe Ox) : $S_{CB} = 0$.
    • AB : Surface latérale de cône (rayon a, génératrice $L_{AB}$) $\implies S_{AB} = \pi a L_{AB} = \pi a \sqrt{a^2+b^2}$.
  • Aire totale générée : $S_{/\Delta, Ox} = S_{CA} + S_{CB} + S_{AB} = \pi a^2 + 0 + \pi a \sqrt{a^2+b^2} = \pi a (a + \sqrt{a^2+b^2})$.
  • Application de Guldin : $$ y_G = \frac{S_{/\Delta,Ox}}{2\pi L} = \frac{\pi a (a + \sqrt{a^2+b^2})}{2\pi (a + b + \sqrt{a^2+b^2})} = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$

    Note : Ce calcul est identique à celui pour $y_{G1}$ dans l'exercice précédent, ce qui est logique car la translation était horizontale.

Résultat Final

Les deux méthodes donnent le même centre d'inertie G pour le triangle linéique décalé :

$$ x_G = c + \frac{b (b + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$ $$ y_G = \frac{a (a + \sqrt{a^2+b^2})}{2 (a + b + \sqrt{a^2+b^2})} $$ $$ z_G = 0 $$

Conclusion

La méthode par translation est nettement plus simple et efficace lorsque la géométrie est une simple translation d'une forme dont le centre d'inertie est déjà connu. La méthode directe par Guldin, bien que plus laborieuse pour $x_G$ ici, confirme les résultats.
