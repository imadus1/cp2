Solution Figure 7.1.4 : Corps linéique (Segment + Demi-cercle)

FIGURE 7.1.4 - Corps linéique : segment [ab] et demi-cercle (adb)

Méthode et Principes

  • Objectif : Trouver le centre d'inertie $G=(x_G, y_G, z_G)$ du corps linéique (densité $\lambda$ constante).
  • Méthode : Premier Théorème de Guldin.
    • L'aire $S_{/\Delta}$ générée par la rotation d'une ligne de longueur $L$ autour d'un axe $\Delta$ (ne la coupant pas ou la touchant) est $S_{/\Delta} = L \times (2\pi R_G)$.
    • $R_G$ est la distance de G à $\Delta$. Donc, $R_G = S_{/\Delta}/(2\pi L)$.
  • Planarité : La figure est dans le plan $z=0$, donc $z_G=0$.
  • Composition : Le centre d'inertie global $G$ est le barycentre des centres d'inertie des composants ($G_{seg}$, $G_{semi}$), pondérés par leurs longueurs respectives.

Géométrie et Longueurs

  • Segment [ab] :
    • Extrémités : a(R, -r, 0) et b(R, r, 0).
    • Description : Segment vertical situé à l'abscisse $x=R$.
    • Longueur : $L_{seg} = 2r$.
  • Demi-cercle (adb) :
    • Description : Arc de cercle centré en c(R, 0, 0) de rayon r.
    • Paramétrisation : $x(\theta) = R + r\cos\theta$, $y(\theta) = r\sin\theta$, pour $\theta \in [-\pi/2, \pi/2]$.
    • Longueur : $L_{semi} = \pi r$.
  • Longueur totale du corps : $L_{tot} = L_{seg} + L_{semi} = 2r + \pi r = r(2+\pi)$.

Centres d'Inertie des Composants

1. Demi-cercle (adb) : $G_{semi} = (x_{G,semi}, y_{G,semi}, 0)$

  • Coordonnée $y_{G,semi}$ :
    • Par symétrie par rapport à l'axe $y=0$ (passant par c et d) : $y_{G,semi} = 0$.
    • Confirmation par Guldin (rotation autour de $y=-2r$) :
      • Distance d'un point $(x(\theta), y(\theta))$ de l'arc à l'axe $y=-2r$ est $y(\theta) - (-2r) = r\sin\theta + 2r$. Élément de longueur $ds=rd\theta$.
      • Surface générée : $$ S_{/y=-2r, semi} = \int_{-\pi/2}^{\pi/2} (2\pi \times \text{distance}) ds = \int_{-\pi/2}^{\pi/2} 2\pi (r\sin\theta + 2r) (r d\theta) $$ $$ S_{/y=-2r, semi} = 2\pi r^2 \int_{-\pi/2}^{\pi/2} (\sin\theta + 2) d\theta = 2\pi r^2 [-\cos\theta + 2\theta]_{-\pi/2}^{\pi/2} = 4\pi^2 r^2 $$
      • Application de Guldin : La distance du centre d'inertie $G_{semi}$ à l'axe $y=-2r$ est $R_G = y_{G,semi} - (-2r) = y_{G,semi} + 2r$. $$ S_{/y=-2r, semi} = L_{semi} \times (2\pi R_G) = (\pi r) \times 2\pi (y_{G,semi} + 2r) $$
      • $4\pi^2 r^2 = 2\pi^2 r (y_{G,semi} + 2r) \implies y_{G,semi} + 2r = \frac{4\pi^2 r^2}{2\pi^2 r} = 2r \implies y_{G,semi} = 0$.
  • Coordonnée $x_{G,semi}$ :
    • Rotation autour de l'axe Oy (x=0). La distance d'un point de l'arc à Oy est $x(\theta) = R + r\cos\theta$.
    • Surface générée : $$ S_{/Oy, semi} = \int_{-\pi/2}^{\pi/2} (2\pi \times \text{distance}) ds = \int_{-\pi/2}^{\pi/2} 2\pi (R + r\cos\theta) (r d\theta) $$ $$ S_{/Oy, semi} = 2\pi r \int_{-\pi/2}^{\pi/2} (R + r\cos\theta) d\theta = 2\pi r [R\theta + r\sin\theta]_{-\pi/2}^{\pi/2} $$ $$ S_{/Oy, semi} = 2\pi r [ (R\frac{\pi}{2} + r) - (-R\frac{\pi}{2} - r) ] = 2\pi r [R\pi + 2r] = 2\pi^2 Rr + 4\pi r^2 $$
    • Application de Guldin : La distance du centre d'inertie $G_{semi}$ à l'axe Oy est $R_G = x_{G,semi}$. $$ S_{/Oy, semi} = L_{semi} \times (2\pi R_G) = (\pi r) \times (2\pi x_{G,semi}) = 2\pi^2 r x_{G,semi} $$
    • $2\pi^2 Rr + 4\pi r^2 = 2\pi^2 r x_{G,semi} \implies x_{G,semi} = \frac{2\pi^2 Rr + 4\pi r^2}{2\pi^2 r} = R + \frac{4\pi r^2}{2\pi^2 r} = R + \frac{2r}{\pi}$.
    • (Vérification rapide : rotation autour de $x=R$. Distance $r\cos\theta$. $S_{/x=R, semi} = \int 2\pi(r\cos\theta)rd\theta = 4\pi r^2$. Guldin : $(\pi r) \cdot 2\pi(x_{G,semi}-R) = 4\pi r^2 \implies x_{G,semi}-R = 2r/\pi$, cohérent).

Donc, $G_{semi} = (R + \frac{2r}{\pi}, 0, 0)$.

2. Segment [ab] : $G_{seg} = (x_{G,seg}, y_{G,seg}, 0)$

  • Par symétrie (milieu du segment vertical [a(R,-r,0), b(R,r,0)]) : $G_{seg} = (R, 0, 0)$.
  • Confirmation de $x_{G,seg}$ par Guldin (rotation autour de Oy) :
    • Le segment est à une distance constante $R$ de Oy.
    • $S_{/Oy, seg} = (\text{Circonférence}) \times (\text{Hauteur}) = (2\pi R) \times (2r) = 4\pi Rr$.
    • Application de Guldin : $S_{/Oy, seg} = L_{seg} \cdot 2\pi x_{G,seg}$.
    • $4\pi Rr = (2r) \cdot 2\pi x_{G,seg} \implies x_{G,seg} = R$.
  • $y_{G,seg}=0$ par symétrie du segment par rapport à l'axe $y=0$.

Donc, $G_{seg} = (R, 0, 0)$.

Centre d'Inertie Global $G = (x_G, y_G, 0)$

  • Par barycentre des centres d'inertie, pondérés par les longueurs : $G = \frac{L_{seg} G_{seg} + L_{semi} G_{semi}}{L_{tot}}$.
  • Coordonnée $x_G$ : $$ x_G = \frac{(L_{seg} \cdot x_{G,seg}) + (L_{semi} \cdot x_{G,semi})}{L_{tot}} $$ $$ x_G = \frac{(2r)(R) + (\pi r)(R + \frac{2r}{\pi})}{r(2+\pi)} = \frac{2rR + \pi r R + (\pi r)(\frac{2r}{\pi})}{r(2+\pi)} $$ $$ x_G = \frac{2rR + \pi r R + 2r^2}{r(2+\pi)} = \frac{r(2R + \pi R + 2r)}{r(2+\pi)} $$ $$ x_G = \frac{R(2+\pi) + 2r}{2+\pi} = R + \frac{2r}{2+\pi} $$
  • Coordonnée $y_G$ : $$ y_G = \frac{(L_{seg} \cdot y_{G,seg}) + (L_{semi} \cdot y_{G,semi})}{L_{tot}} = \frac{(2r)(0) + (\pi r)(0)}{r(2+\pi)} = 0 $$

Résultat Final

Le centre d'inertie G du corps linéique complet a pour coordonnées :

$$ x_G = R + \frac{2r}{2+\pi} $$ $$ y_G = 0 $$ $$ z_G = 0 $$

Le centre d'inertie se trouve sur l'axe $y=0$, légèrement décalé vers les $x$ positifs par rapport à $R$ (centre du demi-cercle), ce qui est cohérent avec la géométrie (le demi-cercle "tire" le centre de masse vers les $+x$).
